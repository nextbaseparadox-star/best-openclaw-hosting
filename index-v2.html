Buyer's Guide

What to Look for in OpenClaw Hosting

New to hosting? Here's what actually matters — and what doesn't.

⏱️ Uptime

Your agent can't respond if the server is down. Look for 99.9%+ uptime. That's the difference between 9 hours of downtime/year (99.9%) and 87 hours (99%).

⚡ Response Time

Gateway TTFB (time to first byte) measures how fast the server processes your request before calling the AI model. Under 150ms is good. Over 300ms feels sluggish.

🔒 Data Privacy

Your agent has access to conversations, memories, and connected tools. Make sure the host doesn't read, share, or train on your data. Self-hosted and Ampere both pass this test.

🔄 Auto Updates

OpenClaw ships updates frequently. On managed hosting, updates happen automatically. On self-hosted, you need to SSH in and update manually — or set up a cron job.

💾 Backups

Your agent's memory, config, and skills are stored in files. If the server dies without backups, you lose everything. Managed platforms include backups. Self-hosted, you set them up.

📊 Specs (CPU/RAM)

OpenClaw needs at least 2GB RAM and 1 vCPU. For comfortable performance with multiple channels and skills, 4GB RAM and 2 vCPU is the sweet spot.

How to Switch

How to Migrate Your OpenClaw Between Hosts

OpenClaw is fully portable. Here's how to move between any hosting option in ~10 minutes.

1

Back up your data

Copy ~/.openclaw/ — this contains your config (openclaw.json), workspace (memory, skills, SOUL.md), and session data. One folder = your entire agent.

2

Set up the new host

Install OpenClaw on your new server (or sign up for Ampere). Run openclaw onboard to get the base config.

3

Restore your data

Copy your backed-up ~/.openclaw/ folder to the new host. Overwrite the default config and workspace. Your agent's personality, memory, and skills come with it.

4

Update channel configs

If your domain or webhook URLs changed, update your channel configs (Discord bot, Telegram webhook, etc.) to point to the new host.

5

Verify

Send a test message. Your agent should respond with full memory and context intact. Done in ~10 minutes.

FAQ

Frequently Asked Questions

What are the minimum requirements to host OpenClaw?
You need at least 1 vCPU, 2GB RAM, and Node.js 22.14+ (Node 24 recommended). Any modern Linux distribution works. For comfortable performance, we recommend 2 vCPU and 4GB RAM.
Is self-hosting OpenClaw free?
OpenClaw is free and open source (MIT license). You pay for the server and AI model API keys (Anthropic, OpenAI, etc.). With Ampere's free tier, both the hosting platform and the server are free — you just bring your own API key.
What is the best OpenClaw hosting for beginners?
Ampere.sh. One-click setup, no technical knowledge required, free tier available. Our non-technical tester had it running in under a minute. Self-hosted options like Hetzner require Linux command-line experience.
Can I migrate from self-hosted to Ampere (or vice versa)?
Yes. OpenClaw stores everything in a standard folder (~/.openclaw/). Copy it between hosts and your agent, memory, skills, and config come with you. Takes about 10 minutes.
Is my data safe on managed hosting like Ampere?
Yes. Each Ampere user gets a fully isolated private environment. The platform cannot read your conversations, files, or memory. No data sharing, no model training on your data. The platform has been professionally security audited.
Do I need my own AI API key?
Yes, regardless of hosting. OpenClaw connects to model providers (Anthropic, OpenAI, Google) using your API key. You control model selection and AI costs independently from hosting costs.
How does OpenClaw hosting compare to ChatGPT Plus?
Different category entirely. ChatGPT Plus is a chat interface. OpenClaw is an autonomous agent platform — it connects to your tools, takes real actions, runs 24/7, has persistent memory, and works across WhatsApp, Discord, Telegram, etc. Hosting OpenClaw gives you a permanent AI employee, not just a chat window.
Can I run Docker instead of a native install?
Yes. OpenClaw has a Docker option. It adds a layer of isolation but also complexity. For self-hosted VPS, the native install via curl or npm is simpler and slightly faster. Managed platforms like Ampere handle the runtime for you.
What if I outgrow the free tier?
Upgrade to a paid plan or migrate to self-hosted. OpenClaw is portable — you're never locked in. Most users find the free tier sufficient for personal use.
Is Hetzner or DigitalOcean better for OpenClaw?
Hetzner, hands down. Better specs for less money — their CX22 (2 vCPU, 4GB RAM) at €4.51/mo outperforms DigitalOcean's $12/mo droplet. Unless you have DO credits or already use their platform, Hetzner wins on value.
