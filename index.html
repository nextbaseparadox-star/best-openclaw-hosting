Setup Guide

How to Set Up OpenClaw - Self-Hosted vs. Managed

The real cost of self-hosting isn't money - it's your time.

🔧 Self-Hosted VPS (e.g. Hetzner)

~12 steps · 30-60 minutes · Linux required
  1. 1Create a cloud account and add billing
  2. 2Provision a VPS (pick region, OS, plan)
  3. 3SSH into the server as root
  4. 4Update system packages
  5. 5Install Node.js 22+ via nvm or nodesource
  6. 6Install OpenClaw via npm globally
  7. 7Run initial OpenClaw configuration wizard
  8. 8Configure your AI API key in config files
  9. 9Set up a systemd service for auto-restart
  10. 10Configure firewall rules (ufw)
  11. 11Set up a domain / reverse proxy (nginx)
  12. 12Configure SSL (certbot / Let's Encrypt)

~3 steps · 60 seconds · No Linux needed
  1. 1Create a free Ampere.sh account
